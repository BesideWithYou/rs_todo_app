## A Rust Todo app

## How to run?

Clone it to your local

```bash
git clone https://github.com/BesideWithYou/rs_todo_app.git
```

then run it from `cargo run `

## commands

create a todo item: `cargo run create washing`
edit a todo item: `cargo run edit washing`
get a todo item: `cargo run get washing`
delete a todo item: `cargo run delete washing`
